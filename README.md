# Pikmin Handbook | 皮克敏圖鑑與純點搜尋

這是一個專為 **Pikmin Bloom** 玩家設計的精美工具網站，致力於整合明信片蒐集與飾品皮克敏（Decor Pikmin）的純點座標搜尋。

👉 **[立即查看發布網站](https://iamernie8199.github.io/Pikmin-Flying-Handbook/)**

---

## 🌟 主要功能

### 1. Pikmin 明信片圖鑑 (Postcard Explorer)
- **全球資料**：涵蓋來自 80+ 個國家、超過 900 張精美明信片。
- **靈活篩選**：支援依據 **國家、類型、分類與標籤** 進行過濾。
- **視覺呈現**：採用現代化的瀑布流 (Masonry) 佈局，讓探索明信片如同翻閱實體相簿。

### 2. 飾品皮克敏純點搜尋 (Pure Spot Search)
- **精準定位**：提供各種皮克敏飾品所需的「純點」座標。
- **豐富分類**：內建 30+ 種分類（如電影院、公車站、拉麵店、甜點店等）。
- **一鍵複製**：整合剪貼簿功能，點擊按鈕即可快速複製 GPS 座標，直接在遊戲中使用。

---

## 🛠️ 技術架構

- **前端**：Vanilla HTML5, CSS3, JavaScript (ES6+)。
- **美學設計**：採用磨砂玻璃 (Glassmorphism) 風格，注重視覺品質與響應式開發。
- **資料儲存**：使用靜態 JSON 資料來源 (`merged_data.json`, `pure.json`)，確保系統極速加載。
